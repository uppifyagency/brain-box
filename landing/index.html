Come funziona

Come funziona un cervello aziendale self-hosted?

Una cartella in entrata, un connettore in uscita: trascini file dentro, fai domande fuori — tutto sul tuo server.

01

Trascina

Ogni dipendente ha una cartella "Azienda Brain" sul Desktop, divisa per reparti. Un watcher carica i file al volo sulla vostra rete privata (Tailscale): PDF, Word, Excel, scansioni, foto di documenti, audio, video.

02

Diventa conoscenza

Il box converte tutto in markdown versionato in git, lo indicizza con embedding vettoriali e un LLM locale (Gemma via Ollama) estrae i fatti chiave: date, impegni, cifre. OCR e trascrizione inclusi, on-device.

03

Interroga

«Quando scade il contratto con X?» — dal proprio Claude Code, Codex o Gemini via MCP, con risposte citate e permessi per reparto. Le domande si fanno in italiano normale.

Suite di test notturna

6 test end-to-end girano ogni notte in cron: se qualcosa si rompe, il box scrive da solo il referto. Prima notte in piena autonomia: verde.

Guardian self-healing

Ogni 15 minuti un guardiano deterministico sorveglia container, disco, modelli e code — e cura i guasti noti con azioni idempotenti da runbook.

Backup collaudato

Backup notturno con restore provato davvero su database di scarto. Un backup mai ripristinato è una speranza, non un backup.

Trappole già pagate

La documentazione include la tabella di troubleshooting dei bug reali già risolti sul campo — con causa e cura, perché tu non debba riscoprirli.

Permessi per reparto

Ogni dipendente ha un token personale: legge i reparti a cui ha accesso, scrive solo nel suo. Il perimetro è il token, la revoca è immediata.

Zero lock-in

Il system of record è markdown in git. Il database vettoriale si ricostruisce con un comando. Puoi andartene quando vuoi — è il punto.

FAQ

Domande frequenti

È davvero gratis?

Sì: la fascia Always Free di Oracle Cloud include una VM ARM 4 OCPU / 24 GB che non scade mai, Tailscale è gratuito fino a 100 dispositivi e i modelli AI sono open weight ed eseguiti sulla tua VM. L'unico costo reale è l'ora che serve per il deploy.

I miei dati sono al sicuro?

I dati vivono in markdown versionato in git sul tuo server, raggiungibile solo dentro la tua rete privata Tailscale. L'intelligenza artificiale è un LLM locale (Gemma via Ollama): nessun documento viene mai inviato a un modello cloud. Ogni accesso passa da un token personale con permessi per reparto.

Che formati di file supporta?

10 formati: PDF anche scansionati (con lettura ottica locale), Word, Excel, PowerPoint, testo, markdown, epub, csv, foto di documenti come fatture e contratti, audio e video (trascritti automaticamente).

Serve essere sviluppatori per usarlo?

Per il deploy serve seguire una guida tecnica di circa un'ora (documentata passo-passo). Per l'uso quotidiano no: i dipendenti trascinano file in una cartella sul Desktop e fanno domande in italiano al loro assistente AI.

Con quali assistenti AI funziona?

Con qualsiasi agente che parli MCP (Model Context Protocol) via HTTP: Claude Code, claude.ai, Codex CLI, Gemini CLI. Le ricette di collegamento sono incluse nel repository.
